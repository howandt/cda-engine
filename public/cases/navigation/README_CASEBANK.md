## ✨ Fordele ved den nye struktur

### 1. **Hurtig navigation**
- Find præcis den type case du søger
-Spring direkte til relevant diagnose
- Se cases i den rigtige kontekst (skole/børnehave/hjem)

### 2. **Nemmere vedligeholdelse**
- Tilføj nye cases i den rigtige fil
- Mindre filer = hurtigere at arbejde med
- Klar struktur = færre fejl

### 3. **Bedre overblik**
- Se hvor mange cases du har af hver type
- Identificer huller i din case-samling
- Planlæg nye cases strategisk

### 4. **Professionel præsentation**
- Når du viser det til Specialisterne
- Når du licenserer til skoler
- Når lærere skal finde relevante cases

## 🚀 Sådan bruger du filerne

### For at finde en specifik case:

**Metode 1: Via diagnose**
1. Åbn `CDA_Cases_OVERSIGT.md`
2. Find diagnose-kategorien
3. Åbn den relevante fil (f.eks. `CDA_Cases_ADHD.md`)
4. Scroll til miljø-sektionen (SKOLE/BØRNEHAVE/HJEM)

**Metode 2: Via indeks**
1. Åbn `CDA_Cases_INDEKS.md`
2. Søg efter case-nummer eller navn (Ctrl+F / Cmd+F)
3. Se hvilken fil casen er i
4. Åbn filen og find casen

**Metode 3: Via tema**
1. Åbn `CDA_Cases_INDEKS.md`
2. Se under "🔍 Sådan finder du den rigtige case"
3. Find dit tema (f.eks. "Mobning", "Lægebesøg", "Frisør")
4. Gå til de angivne case-numre

## ➕ Sådan tilføjer du nye cases

### Trin 1: Vælg den rigtige fil
Spørg dig selv:
- Hvad er **primær diagnosen**? → Vælg fil
- Hvilket **miljø** foregår det i? → Vælg sektion

### Trin 2: Find den rigtige sektion
Hver diagnose-fil har 3 sektioner:
```
## 🏫 SKOLE CASES
## 🏠 BØRNEHAVE CASES  
## 🏡 HJEM CASES
```

### Trin 3: Tilføj din case
1. Kopier en eksisterende case som skabelon
2. Ret case-nummer (f.eks. næste ledige nummer)
3. Udfyld alle felter
4. Gem filen

### Trin 4: Opdater statistik
I toppen af filen:
- Opdater total antal cases
- Opdater antal cases per miljø

### Trin 5: Opdater OVERSIGT og INDEKS
Når du har tilføjet flere cases, opdater:
- `CDA_Cases_OVERSIGT.md` (total antal)
- `CDA_Cases_INDEKS.md` (tilføj nye cases til listen)
